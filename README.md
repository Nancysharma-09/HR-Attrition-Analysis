# HR Attrition Prediction Project

## 📊 Overview
This project predicts employee attrition using Machine Learning.

## 📂 Dataset
IBM HR Analytics Dataset (1470 rows, 35 columns)

## 🎯 Objective
To identify employees likely to leave the company.

## 🛠 Tools Used
Python, Pandas, Matplotlib, Seaborn, Scikit-learn

## 🔍 Process
- Data Cleaning
- EDA
- Feature Encoding
- Logistic Regression Model
- Evaluation (Accuracy, Confusion Matrix)

## 📈 Result
Model predicts employee attrition with good accuracy.
